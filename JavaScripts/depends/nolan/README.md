然而它亦面临无法避免的难题：

- **乳臭未干** Nolan 目前处于极度幼稚的状态。
- **屋檐之下** Nolan 高度依赖现有的 MetaWorld Camera 系统，作者无法提供随时对接更新的承诺，亦无法提供超越框架功能的承诺。
