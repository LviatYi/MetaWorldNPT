阅读该文档时，推荐安装以下字体：

- [JetBrainsMono Nerd Font Mono][JetbrainsMonoNerdFont]

## Functional ⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠐⠒⠒⠒⠒⠚⠛⣿⡟⠄⠄⢠⠄⠄⠄⡄⠄⠄⣠⡶⠶⣶⠶⠶⠂⣠⣶⣶⠂⠄⣸⡿⠄⠄⢀⣿⠇⠄⣰⡿⣠⡾⠋⠄⣼⡟⠄⣠⡾⠋⣾⠏⠄⢰⣿⠁⠄⠄⣾⡏⠄⠠⠿⠿⠋⠠⠶⠶⠿⠶⠾⠋⠄⠽⠟⠄⠄⠄⠃⠄⠄⣼⣿⣤⡤⠤⠤⠤⠤⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄⠄

它旨在提供如下便利：

- [ ] **呼风唤雨** 模糊搜索与拼音搜索 随心所欲地触及.
- [ ] **熟能生巧** 基于热力的命令推荐. 优先推荐最近使用的命令.
- [ ] **一板一眼** 简单类型与 Mw 常用类型参数验证、Enum 自动转下拉列表与配置表参数快速预览.
- [x] **前辙可鉴** 提供命令与参数的历史记录 快速重复命令.
- [ ] **召之即来** 提供常用的一般性命令.
- [x] **交互艺术** 允许拖动 显隐 窗口.
- [x] **触手可得** 自生成 UI. 不依赖任何其他要素.
- [x] **一夫当关** 权限认证. KV 存储的用户权限凭证.
